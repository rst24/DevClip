Getting Started
To use AI-powered features, you need an API key from the DevClip web app.

1. Go to the DevClip Dashboard
2. Login with Google, GitHub, Apple, or X
3. Navigate to Settings → API Keys tab
4. Click "Generate Key" and copy it
5. Paste the key below and click Save
Your API key is stored securely and only used to authenticate with the DevClip API
Choose Production for deployed app, Local for development
